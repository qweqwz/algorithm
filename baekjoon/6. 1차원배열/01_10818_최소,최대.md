1. 문제

   N개의 정수가 주어진다. 이때, 최솟값과 최댓값을 구하는 프로그램을 작성하시오.

2. 입력

   첫째 줄에 정수의 개수 N (1 ≤ N ≤ 1,000,000)이 주어진다. 둘째 줄에는 N개의 정수를 공백으로 구분해서 주어진다. 모든 정수는 -1,000,000보다 크거나 같고, 1,000,000보다 작거나 같은 정수이다.

3. 출력

   첫째 줄에 주어진 정수 N개의 최솟값과 최댓값을 공백으로 구분해 출력한다.

4. 코드

   4-1.
   
   ```c++
#include <iostream>
   using namespace std;
   int main(){
       int N, a;
       cin>>N;
       int arr[N];
       for (int i=0;i<N;i++){
           cin>>a;
           arr[i]=a;
       }
   
       int max=arr[0];
       int min=arr[0];
       for (int j=0; j<N;j++){
           if (max<=arr[j]){
               max=arr[j];
           }
           if (min>=arr[j]){
               min = arr[j];
           }
   
       }
       cout<<min<<" "<<max;
       return 0;
   }
   ```
